Installation

 

  1. Unzip the VideoGall plugin in the plugins folder in wp-content directory i.e "wp-content/plugins" or Go to Plugins section -> Add New -> Upload Plugin
  2. Go to your admin page and plugins section and activate the VideoGall plugin

Adding Videos

  1. Copy the URL of the video page from the browser
  2. Go to VideoGall Settings page
  3. Go to Add new Video section
  4. Paste the URL in the Add Video URL field
  5. VideoGall automatically fetches the thumbnail of the video. Incase you want to use your own custom thumbnail, use the upload button next to Add Video Thumbnail field to upload thumbnail
  6. To pick category for the video, you will have to add new categories first in the Video Categories section
  7. Hit Add New Video button

Adding Categories

  1. Go to Videogall Settings
  2. Go to Video Categories section
  3. Here you can add,update or remove categories

Editing Videos

  1. To edit Videos,
  2. Go to Videogall Settings Page
  3. Click edit for the video you want to edit
  4. Note: If you modified the URL of a video, it won't refresh the thumbnail automatically. You will have to remove the thumbnail URL and then save the video in order to get the new thumbnail for the new URL

Usage

  1. Go to Post or Page Editor
  2. Use the Shortcode button for videogall and it will enter [myvideogall:all] . This will show all the videos.
  3. In order to show videos from certain category, replace "all" with the name of category e.g. [myvideogall:movies]. Note: category name is case-insensitive
  4. To show videos from multiple categories, use comma separated categories. Eg. [myvideogall:movies,news]
